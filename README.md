# AquaGuard SL 🌍💧
**Clean Water, Healthy Life**

AquaGuard SL is a web and mobile application platform designed to digitize water issue reporting and management in Sierra Leone. It empowers citizens, WASH officers, NGOs, and health workers to collaboratively address water challenges through real-time reporting, transparent tracking, and coordinated resolution.

---

## 🚨 Problem Statement
Sierra Leone faces one of the lowest rates of access to safe drinking water in West Africa.  
- Less than 60% of the population has access to basic water services.  
- Only 15% have access to safely managed drinking water.  
- Issues include contaminated sources, broken infrastructure, pipeline leakages, and poor sanitation.  

The existing paper-based reporting system is slow, prone to data loss, and disconnected from response teams. AquaGuard SL solves this by creating a **direct digital channel** between communities and authorities.

---

## 🎯 Project Objectives
- Real-time water issue reporting via mobile devices  
- GPS and photo-verified evidence for validation  
- Dashboards for WASH officers and local authorities  
- Alerts for contamination and service disruptions  
- Analytics to guide infrastructure investment  
- Alignment with **UN SDG 6 (Clean Water and Sanitation)**  

---

## 👥 Target Users
- **Community Users (Citizens/Residents):** Report issues and receive alerts  
- **Local Authority / WASH Officers:** Verify and resolve issues via web portal  
- **NGO / Health Workers:** Monitor health impacts and coordinate responses  
- **System Administrators:** Manage platform, users, and data  

---

## ⚙️ Key Features
- 📍 Real-time issue reporting with GPS + photo uploads  
- 🗺️ Interactive water map showing hotspots and resolution status  
- 🔔 Automated alerts for contamination and updates  
- 👤 Role-based access control  
- 📊 Analytics dashboard with charts, heatmaps, and exportable reports  
- 🤝 Response coordination tools for NGOs and government agencies  
- ⭐ Community feedback and rating system  

---

## 🏗️ System Architecture
AquaGuard SL follows a **Three-Tier Architecture**:
1. **Presentation Layer (UI):** Mobile app (React Native) + Web portals  
2. **Application Layer (Business Logic):** Node.js backend with microservices  
3. **Database Layer (Data Storage):** PostgreSQL + AWS RDS + S3 for photo evidence
